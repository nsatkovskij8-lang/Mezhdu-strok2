# Между строк

Личный дневник с искусственным интеллектом.

## Описание

«Между строк» — это личный дневник с искусственным интеллектом, который помогает человеку замечать закономерности в своих мыслях, эмоциях и жизни.

Он не заменяет психолога и не ставит диагнозы.

Его задача — помочь пользователю лучше понять самого себя.

## Возможности

- 📝 Личный дневник
- 🤖 Анализ записей ИИ
- 📊 Аналитика настроения
- 💡 «Что осталось между строк»
- 🔥 Серии записей
- ☁️ Синхронизация
